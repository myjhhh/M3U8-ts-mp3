# M3U8分片提取转换工具
一个自动化的Linux Shell脚本，可解析本地M3U8文件、提取TS分片、批量转换为MP3格式，并自动清理原TS文件，全程无需手动干预。

## 🌟 核心功能
- 🎯 自动解析本地M3U8文件，提取所有TS分片的完整路径
- 📂 保留TS分片的原始文件名，复制到指定目录
- 🎵 批量将TS文件转换为同名MP3（128k音质，可自定义）
- 🗑️ 转换成功后自动删除原TS文件，仅保留MP3
- 🛠️ 自动检测Linux系统发行版，缺失FFmpeg时自动安装
- ✅ 完善的错误校验（文件存在性、路径合法性、转换结果）
- 📋 详细的日志输出，清晰展示每一步操作结果
